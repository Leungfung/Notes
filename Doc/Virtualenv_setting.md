# Virtualenv 环境设置

## 安装Venv

```shell
 pip3 install virtualenv
```

## 设置Venv

```shell
#建立python3虚拟环境不包含系统组件
virtualenv --no-site-packages -p python3  ~/Dev/py3pytorch
```

## 激活Venv

```shell
cd py3pytorch
source bin/activates
```

## 激活成功

```shell
(py3pytorch)  leo@LeungFungdeMacBook-Pro > ~/Dev/py3pytorch :
```
